# Resumizer

RAG pipeline to analyze Resume Dataset. Analyze various resumes in pdf format.

## Pre-requisite
- Run the script that setups the venv and starts jupyter

```
./startup.sh
```

- Run a local LLM. I ran qwen2.5:3b

## Running notebook
* In the Jupyter notebook select Menu->Kernel->resumerag.ipynb
* Run -> Run All Cells

## Results
* The last cell output has the result
* You may also check rag_evaluation_result.csv for a structured result.
